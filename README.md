# Audit Sémantique Cameroun

Ce package fournit des outils pour l'audit sémantique des lois de finances du Cameroun à l'aide de méthodes NLP et de modèles Transformers.

## Installation

```bash
pip install .
```

## Utilisation

Importez le package dans vos scripts Python :

```python
import audit_semantique
```

## Structure du package
- `audit` : Audit sémantique principal
- `clustering` : Algorithmes de clustering
- `modeling` : Modèles et embeddings
- `preprocessing` : Prétraitement des données
- `stats` : Statistiques et tests
- `topic_modeling` : Modélisation de sujets (LDA)
- `visualization` : Visualisation des résultats
